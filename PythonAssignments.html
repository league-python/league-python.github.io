Hello World - Print out "hello world"

Greeter - Ask the user for their name and print a nice greeting using their name in the response

Word Repeater - Ask the user for a word and a number. Print the word as many times as the number they entered.

        example:

                word: pizza

                number: 5

                pizza

                pizza        

                pizza

                pizza

                pizza

Riddle - Ask the user a riddle. Tell them if they got the correct answer or not

Change Calculator - Ask the user how many pennies, dimes, nickels, and quarters they have. Calculate their total dollar amount and give them the answer.

Test Scorer - Ask the user what score they got on a test and give them a letter grade based on the following chart:

        90 - 100: A

        80 - 89: B

        70 - 79: C

        60 - 69: D

        0 - 59: F

High Low Game - Generate a random number from 1 - 100. Ask the user to guess the number. Tell them if they are too high or too low. Repeat this until they guess the number. Tell them how many tries it took.
